  > What the product says: _"World's first open-source, agentic video production system"_

### 3. How does it find users?
**GitHub Trending** → HN 'Show HN' post → dev Twitter/X. The README *is* the landing page. Stars compound: once it trends, press coverage follows automatically.

**Software developers and engineers** (B2C prosumer / B2B small teams).

_AI Chat for Search helps software developers and engineers by letting them display ai chat response alongside search engine results._
